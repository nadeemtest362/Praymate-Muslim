## Design System Integration

### Colors
- Lock State: `#FF6B8B` (Warning Red)
- Unlock State: `#8BED4F` (Success Green)  
- Emergency: `#FFA500` (Orange)
- Stats/Progress: `#8B80F9` (Brand Purple)

### Typography
- Headers: Font weight 900, -1.1 letter spacing
- Body: Font weight 400-600
- Scripture: Italic, slightly smaller

### Animations
- Lock bounce when trying to open
- Smooth slide-up for lock screen
- Confetti on unlock (subtle)
- Progress bar fills on stats

### Haptics
- Heavy impact on lock attempt
- Success haptic on unlock
- Light tap on emergency unlock
